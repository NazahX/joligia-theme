# Lumera Jewellery Theme

Premium Shopify Online Store 2.0 theme scaffolded for a small, curated jewellery launch and designed to scale cleanly as the catalog grows.

## Replace First

- Upload your real logo in `Theme settings > Logo`.
- Replace homepage imagery in the homepage sections inside the Shopify customizer.
- Update placeholder brand copy in the customizer for homepage sections, footer, About page, FAQ page, and announcement bar.
- Connect menus in the header and footer section settings.

## Recommended Customizer Pass

1. Set `Heading font` and `Body font` if you want different Shopify-safe fonts.
2. Adjust `Logo width` so the mark feels balanced on desktop and mobile.
3. Review `Announcement bar`, `Header`, `Hero`, `Featured collection`, `Testimonials`, `Newsletter`, and `Footer`.
4. Assign a featured collection for launch so the small catalog feels intentional.
5. Populate product metafields or descriptions later if you want richer PDP storytelling.

## Theme Structure

- `layout/theme.liquid`: global shell and shared assets.
- `config/settings_schema.json`: global theme settings.
- `sections/`: homepage, commerce, and page-builder sections.
- `snippets/`: shared icons, pricing, cards, and responsive media rendering.
- `templates/`: OS 2.0 JSON templates for home, product, collection, cart, search, and pages.
- `assets/base.css`: design system and responsive styling.
- `assets/theme.js`: minimal JS for drawers, sticky header, gallery, and AJAX cart.

## Notes

- The theme is intentionally built around whitespace, editorial composition, and curated merchandising so seven products still feel premium.
- Product cards and collection layouts are already structured to scale to larger inventories without redesign.
